# Техническое задание на проект: Планировщик Путешествий

## 1. Цель проекта

Создание веб-сервиса для планирования путешествий с динамическим расчетом бюджета. Система помогает путешественникам оценить стоимость поездки в реальном времени с учетом актуального курса валют.

## 2. Роли пользователей

- **Гость**: Просмотр каталога направлений, просмотр информации о стоимости жизни.
- **Зарегистрированный пользователь**: Создание и сохранение собственных бюджетов, управление списком расходов.

## 3. Модели данных

### Модель 1: Destino (Направление)
- `nombre` (CharField): Название города
- `pais` (CharField): Страна
- `moneda_local` (CharField): Код местной валюты (EUR, USD, и т.д.)
- `costo_vida_diario` (DecimalField): Ориентировочная стоимость жизни за день
- `descripcion` (TextField): Описание направления

### Модель 2: PresupuestoViaje (Бюджет поездки)
- `usuario` (ForeignKey): Пользователь, создавший бюджет
- `destino` (ForeignKey): Выбранное направление
- `fecha_creacion` (DateTimeField): Дата создания
- `dias_estimados` (PositiveIntegerField): Количество дней
- `moneda_usuario` (CharField): Валюта пользователя
- `tasa_cambio_usada` (DecimalField): Использованный курс обмена
- `costo_total_estimado` (DecimalField): Общая стоимость

### Модель 3: ItemGasto (Статья расхода)
- `presupuesto` (ForeignKey): Бюджет, к которому относится расход
- `categoria` (CharField): Категория расхода (перелет, проживание, еда и т.д.)
- `descripcion` (CharField): Описание расхода
- `costo_estimado` (DecimalField): Ориентировочная стоимость

## 4. Ключевой функционал

- Просмотр списка направлений с сортировкой по стоимости
- Детальная страница каждого направления с информацией о расходах
- Форма для планирования поездки с выбором направления, дней и валюты
- Расчет бюджета с конвертацией валюты в реальном времени через API
- Сохранение бюджетов для авторизованных пользователей

## 5. Внешние интеграции и аналитика

- **Интеграция**: Использование ExchangeRate-API для получения актуальных курсов валют в реальном времени.
- **Аналитика**: Использование библиотеки Pandas для анализа сохраненных бюджетов и вывода статистики (самые популярные направления, средняя стоимость поездки).
